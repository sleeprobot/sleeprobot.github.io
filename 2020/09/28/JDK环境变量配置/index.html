JDK环境变量配置

Java SE Development Kit简称JDK，学习Java开发，第一步当然是需要配置JDK环境变量了，首先，需要下载JDK，这里以JDK8为例，操作系统环境为win10。

下载JDK

官方下载地址：https://www.oracle.com/java/technologies/javase-downloads.html

点击JDK Download后，跳转到新页面，找到Windows安装程序，点击即可下载

根据自己的系统位数，选择相应的安装包下载，这里我下64位（Windows x64）的。

下载时需要Oracle账户，自己用邮箱注册一个登录即可。

安装JDK

下载完JDK安装程序后，双击开始安装。

更改JDK安装位置

JDK默认的安装位置是C:\Program Files\Java\jdk xxx(xxx表示版本号)，这里我装到D盘，你们可以自行选择，但是要记住自己安装在哪，后续配置环境时有用。

点击下一步，安装JDK。

更改JRE安装位置

安装完JDK之后，需要继续安装JRE，默认的安装位置同样在C盘，这里我同样安装到D盘，和JDK放一块，同样可以不做更改，但是要自己能找到安装位置。

修改好安装位置之后，点击下一步，开始安装。

完成安装

看到这个界面，表示安装完成了，点击关闭即可。

关闭自动检查更新（可选）

如果电脑安装了管家类安全软件，在安装完成后会提示jusched.exe进程尝试加入开机自启，这个进程文件会定期检查Java是否有更新，占用电脑的资源不多，如果不需要（其实也没啥更新的），可以关闭自动检查更新。

打开电脑的控制面板，点击程序：

点击Java，会打开Java控制面板：

打开Java 控制面板，选择更新选项卡，将自动检查更新前面的√去掉，选择不检查，点击确定即可：

JDK环境变量配置

  1. 鼠标右击计算机（此电脑）->属性->高级系统设置->高级->环境变量

  2. 在系统变量中，点击新建，新建JAVA_HOME变量，变量的值：JDK的安装目录

  3. 在系统变量中找到Path变量，双击打开，新增两个变量%JAVA_HOME%\bin和%JAVA_HOME%\jre\bin，点击确定

    如果是win7或Xp的系统，要注意每个环境变量后面要加英文的;

  4. 在系统变量中新建CLASSPATH变量，变量值：.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（注意前面的.），点击确定，保存退出。

  5. 检查是否配置成功，win+R输入cmd，然后回车，在黑乎乎的窗口输入java -version，能够看到你安装的jdk版本信息，即环境配置成功。

    如果提示命令未找到，请检查是否漏了上述的哪个步骤。

现在可以开始Java的学习之旅了！
